## Project Summary

This project predicts **telecom customer churn using Machine Learning**. Multiple models, including **Logistic Regression, Random Forest, and Gradient Boosting**, were trained and evaluated.

Among them, **Gradient Boosting performed best with an ROC-AUC score of 0.8436**. The analysis identified **contract type, tenure, and monthly charges** as the key factors affecting churn.

Results showed that customers with **month-to-month contracts, high charges, and low tenure** were more likely to leave. Based on these insights, retention strategies such as **personalized offers and promoting long-term plans** were suggested.
